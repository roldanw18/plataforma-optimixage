🏗️ ARQUITECTURA IMPLEMENTADA
════════════════════════════════════════════════════════════════════════════

FLUJO DE TRADUCCIÓN:
  App (main.jsx)
    ↓ importa './config/i18n'
    ↓
  i18n.js (configuración central)
    - fallbackLng: 'es'
    - supportedLngs: ['es', 'en']
    - Namespaces: [common, auth, admin, client, validation, errors]
    - Detección: localStorage → navigator
    - Persistencia: localStorage
    ↓
  Componentes React
    const { t } = useTranslation('namespace')
    {t('key')}


DETECCIÓN DE IDIOMA (Primera Visita):
  1. ¿localStorage tiene idioma guardado?
     └─ SÍ → Usar ese idioma
  2. ¿Navegador tiene idioma soportado (ES/EN)?
     ├─ SÍ → Usar idioma del navegador + guardar en localStorage
     └─ NO → Usar fallback (ES)
  3. Usuario cambia idioma:
     └─ i18n.changeLanguage('en') → localStorage se actualiza
